# How to Use Claude Code with Qwen models for Free.

## Prerequisites
*   **Node.js (v18+):** Ensure Node.js is installed and working fine.
*   **Qwen CLI:** Installed, authenticated and working fine.
*   **Claude Code:** Installed and working fine
*   **Claude Code Router:** Installed and working fine

---

1. For Detailed Windows Installation: [Install Claude Code on Windows](./windows_setup.md)
2. For Detailed Bash Installation: [Install Claude Code on Bash](./bash_setup.md)

---

## Installing all Prerequisites

### Step 2: Install the Claude Code

Open bash terminal and run the following command to install the Qwen Code

```bash
npm install -g @anthropic-ai/claude-code
```

### Step 3: Install the Qwen Code

Open bash terminal and run the following command to install the Qwen Code

```bash
npm install -g @qwen-code/qwen-code@latest
```

### Step 4: Install the Claude Code Router

Open bash terminal and run the following command to install the Claude Code Router

```bash
npm install -g @musistudio/claude-code-router
```
